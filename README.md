## 🚧 Sobre

**OrçaFácil** é uma ferramenta online para cálculo de materiais e orçamentos voltada para construção civil.  
Facilita o planejamento e controle de custos, ajudando profissionais e clientes a tomarem decisões mais rápidas e precisas.

## 🚀 Começando

### Pré-requisitos

- Node.js v14 ou superior  
- npm (gerenciador de pacotes)

> Recomendamos usar [nvm](https://github.com/nvm-sh/nvm) para gerenciar versões do Node.js

### Instalação

1. Clone o repositório:
   ```bash
   git clone <SEU_GIT_URL>

    Entre na pasta do projeto:

cd <NOME_DO_PROJETO>

Instale as dependências:

npm install

Inicie o servidor de desenvolvimento:

    npm run dev

    Abra no navegador o endereço que aparecer no terminal (geralmente http://localhost:3000)

🛠️ Tecnologias usadas

    Vite - Build tool moderna

    React - Biblioteca para interface

    TypeScript - Superset de JavaScript com tipagem

    shadcn-ui - Biblioteca de componentes UI

    Tailwind CSS - Framework CSS utilitário

📝 Como editar o projeto

Você pode editar o código de várias formas:
1. Online via Lovable

Acesse Lovable Project e edite direto no navegador.
Alterações são commitadas automaticamente.
2. Localmente com sua IDE

Siga os passos do tópico "Começando" para rodar o projeto localmente e editar no seu editor preferido.
3. Direto pelo GitHub

Navegue até o arquivo desejado no repositório e clique no ícone de lápis para editar.
Faça commit direto pelo site.
4. GitHub Codespaces

Abra um Codespace pelo botão Code > Codespaces e edite o projeto diretamente no ambiente online da GitHub.
